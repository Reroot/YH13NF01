2. Student Enrollment

Generate the results of the students enrolled in different grades. Create the following classes:

1. The Student class should have three class variables:

  • a String name
  • an integer studentClass
  • a float result.

It should also contain the following methods:

  • Student(String s, int sClass): It should set the name of the student to s and the class of the student to Class. It should also print "Added student: {name} to the roll of class: {sClass}". For example if s = Harry and sClass = 1, it should print "Added student: Harry to the roll of class: 1".
  • String getName(): It should return the name of the student.
  • String publish(): If the result of the student is greater than or equal to 33.33,  it should print "{name} has been promoted to class: {studentClass +1}", otherwise it should print "{name} has been retained in class: {studentClass}", where {name} denotes the name of the student and {studentClass} denotes the current class of the student.

2. The Result class should inherit the Student class and should have three class variables:

  • Integer subject1
  • Integer subject2
  • Integer subject3

It should also contain the following methods:

  • Result(int a, int b, int c, String s, int sClass):
  1. It should assign the student name = s and studentClass = sClass.
  2. It should assign a to subject1, b to subject 2 and c to subject3.
  3. It should print the following lines: "{name} obtained {marks} marks in subject1", "{name} obtained {marks} marks in subject2" and "{name} obtained {marks} marks in subject3", where {name} denotes the name of the student and {marks} denotes the marks in that subject. {marks} will be equal to subject1, subject2 and subject3 for the respective subjects.
  • String calculateResult():
  1. It should calculate the result of the student using the formula ((subject1 + subject2 + subject3) *100)/300.
  2. It should then call the publish( ) method from the Student class.
  • String changeMarks(int newMarks, String subject):
  1. It should print the following line first, "{name} has ordered a recheck in {subject} ", where name denotes the name of the student and subject denotes the name of the subject, passed as a parameter.
  2. It should assign newMarks to either subject1, subject2 or subject3 depending on the parameter subject. For example, if subject = "subject1", it should assign newMarks to subject1 variable of the Result class.
  3. It should then calculate the result using the calculateResult( ) method of the Result class and return a string in the following format, "Following is the new result: {name} has been promoted  to class: {studentClass + 1}" or "Following is the new result: {name} has been retained in class: {studentClass}", where name is the name of the student and studentClass is the current class of the student.

 

Constraints

  • number of students = 5

 

Input Format For Custom Testing

The first line contains space-separated strings , the names of the students.
The next n (= number of students) lines contains three space-separated integers each, the marks of the ith student in subject1, subject2 and subject3 respectively.

The next line contains n space separated integers, the current class of the students.

The next line contains a string, the subject in which recheck is to be ordered.

The last line contains an integer, the newMarks in the above subject.
